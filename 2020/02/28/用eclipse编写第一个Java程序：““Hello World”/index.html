eclipse编写 “Hello World” Java程序

1. 创建项目

依次点击“File” –> “New” –> “Project”（如下图）

在弹出的“New Project”页面中选择“Java”里面的“Java Project”，然后点击“Next >”

在弹出的“New Java Project”页面中填入项目名称（“Project name”），其它选项默认不做改变，点击“Finish”

接着会弹出如下界面，一般选择不去创建“Module-info.java”，点击“Don’t Create”即可

如上操作，我们就建好了项目

2. 新建Package包

右键项目中的“src”，依次选择“New” –> “Package”

包名并无特殊要求，主要是开发之中约定的规范（详情请查看CSDN上的这篇文章：“java包名（package）命名规则” ）。
这里我们主要是通过该例子知晓并熟练编写一个Java程序，所以通俗点，直接输入“helloworld”，点击“Finish”

3. 新建Class类

右键刚创建好的Package包：“helloworld”，依次选择“New” –> “Class”

填写好类名“helloworld”，勾选下图红框public static void main(String[] args)所对应的勾选框，可以自动形成main（如下第二张图红框中内容）
